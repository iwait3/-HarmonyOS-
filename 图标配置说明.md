# 应用图标配置说明

## 已完成配置

1. **桌面应用图标**：已修改为使用 `mainp.png`
   - 配置文件：`entry/src/main/module.json5`
   - 修改项：`icon` 字段从 `$media:layered_image` 改为 `$media:mainp`

2. **启动页面图标**：已创建专用配置
   - 配置文件：`AppScope/resources/base/media/start_window_icon.json`
   - 配置项：使用 `mainp.png` 作为前景图，背景使用默认背景

## 需要手动处理的图标优化

### 1. 图标尺寸要求

**桌面应用图标推荐尺寸：**
- 主要图标：192x192 像素（用于高分辨率设备）
- 备用尺寸：144x144、96x96、72x72、48x48 像素

**启动页面图标推荐尺寸：**
- 建议尺寸：128x128 像素（不超过256x256）
- 启动图标会按原始尺寸居中显示，建议使用较小尺寸

### 2. 圆角处理建议

**推荐圆角半径：**
- 桌面图标：圆角半径约为图标宽度的20-25%
- 启动图标：圆角半径约为图标宽度的15-20%

### 3. 图标文件准备步骤

1. **准备桌面图标**：
   - 使用图像编辑软件（如Photoshop、GIMP等）
   - 打开 `mainp.png` 文件
   - 调整尺寸为推荐尺寸（192x192像素）
   - 应用圆角效果（半径约38-48像素）
   - 保存为 `mainp.png`

2. **准备启动图标**：
   - 创建新文件，尺寸128x128像素
   - 使用 `mainp.png` 中的图案，适当缩小
   - 应用圆角效果（半径约19-25像素）
   - 保存为 `start_icon.png`
   - 放置在 `AppScope/resources/base/media/` 目录

3. **更新配置文件**：
   - 如果创建了专门的启动图标，修改 `start_window_icon.json`
   - 将 `foreground` 改为 `$media:start_icon`

### 4. 多分辨率支持（可选）

如需支持不同分辨率设备，可在以下目录放置对应尺寸图标：
- `AppScope/resources/phone-ldpi/media/`
- `AppScope/resources/phone-mdpi/media/`
- `AppScope/resources/phone-sdpi/media/`
- `AppScope/resources/phone-xldpi/media/`
- `AppScope/resources/phone-xxldpi/media/`
- `AppScope/resources/phone-xxxldpi/media/`

## 验证配置

配置完成后，请重新编译应用并安装测试，查看图标显示效果是否符合预期。
